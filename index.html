MY PROjects

CS50 WEb development course

This is a course I completed over the 2020 lockdown,  during which I built a large selection of websites. I became experienced with Django, Flask, and HTML/CSS/JS, and became confident in producing websites. Check out the link to my final project github repo and a link to the course below.

Robotron 2084 pygame

During my A level course, I built a remake of the classic arcade game called Robotron, using PyGame in python on the front end, and a flask API and website to drive the logic behind the global user system and leaderboard. I achieved my goals of producing a fun to play, exciting game, to completion, this included writing a full report on the project.

CHess game

Early on in my A Level course I was tasked with creating a chess game, I decided to take this further and created a basic AI using minimax, with alpha beta pruning, I also built a move validator and checkmate enforcement. It was developed using TkInter.

I am the secretary of this association, and the Data Protection Officer for the prom.kim website. In order to qualify for this role, I took a course in DPO skills offered by the University of Derby
